#  Malaria Cell Classification using PyTorch

## Overview

This project uses a Convolutional Neural Network (CNN) built with PyTorch to classify microscopic blood cell images as either **Parasitized** or **Uninfected**.

It demonstrates the complete deep learning workflow, including data preprocessing, augmentation, model training, evaluation and visualization.

---

## Dataset

Cell Images for Detecting Malaria

https://www.kaggle.com/datasets/iarunava/cell-images-for-detecting-malaria

---

## Features

- Custom PyTorch Dataset
- Data Augmentation using Albumentations
- CNN (LeNet-inspired)
- Batch Normalization
- Dropout
- GPU Support
- Binary Classification
- Model Saving
